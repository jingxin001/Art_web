# Carl Cox Inspired Landing Page

This project is a React + TypeScript + Vite rebuild of the referenced Carl Cox homepage structure, using original replacement SVG artwork instead of copied source photography.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run lint`

## Notes

- The layout recreates the overall composition: transparent header, centered brand lockup, hero quote, four equal-height feature panels, slide-out menu, and search overlay.
- Artwork in `public/assets` is local SVG replacement art and can be edited directly if you want to push the visual style closer to the reference.
